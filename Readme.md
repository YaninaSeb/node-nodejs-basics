# Node.js basics

## Use 16.13.0 version of Node.js

## Scripts for testing

**1) File system**
   - npm run fs:create
   - npm run fs:copy
   - npm run fs:rename
   - npm run fs:delete
   - npm run fs:list
   - npm run  fs:read

**2) Command line interface**
   - npm run cli:env
   - npm run cli:args

**3) Modules**
   - npm run modules

**4) Hash**
   - npm run hash

**5) Streams**
   - npm run streams:read
   - npm run streams:write
   - npm run streams:transform

**6) Zlib**
   - npm run zip:compress
   - npm run zip:decompress

**7) Worker Threads**
   - npm run wt:worker

**8) Child Processes**
   - npm run cp
